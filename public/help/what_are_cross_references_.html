What are Cross References?

 Top  Previous  Next

Cross-references are unique identifiers that link the property (BLPU) to external applications - e.g. electoral role, council tax, address-point data. Cross-references are important because of the need to synchronise addresses back to the core systems that were initially used to build the LLPG. It is imperative to keep cross-references accurate as, in some instances, they will be used to ascertain which address records in the external system should be changed. It is usual to find an Electoral Registration, Council Tax and Address-Point cross-reference against all residential properties. Should you find residential properties missing one of these core cross-references this would indicate a data error that you would wish to resolve.
