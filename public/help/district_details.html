District details

 Top  Previous  Next

District details may be changed by an Administrator from the Settings > Lookup tables > Districts and selecting the district you wish to maintain.

Organisation

The name of the organisation.

Name

The name for the district.

ID

The unique district reference number for the district created by the organisation. This should be a numeric value between 1 - 999.

Function

Select the type of function the district is involved in regarding the exchange of Street Works information.

Address

The address for the district.

Postcode

The postcode for the district.

Telephone no.

The telephone number for the district.

Fax No

The fax number for the district.

Permit Scheme ID

The national permit scheme reference.

Closed date

If the district is closed then select a date from the calendar for the date of closure.

 

Make the changes that you require and then click Done.
