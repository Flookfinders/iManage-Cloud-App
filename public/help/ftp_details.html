FTP details

 Top  Previous  Next

FTP details may be changed by an Administrator from the Settings > Lookup tables > Districts and selecting the district you wish to maintain.

FTP server name

The FTP address for receiving batch file transfers.

Server IP address

The IP address for the service of notifications.

FTP directory

The directory name on the FTP Server.

Notification URL

Web service URL for exchange of notifications using XML.

Attachment URL Prefix

The base URL of the FTP site for viewing and downloading of documents that are 'attached' to notifications.

 

Make the changes that you require and then click Done.
