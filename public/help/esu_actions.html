ESU actions

 Top  Previous  Next

The actions for an ESU are found when the ESU tab is selected and there are various places where actions are displayed.

1.At the top of the form.

Action

Description

Expand all

Expand the tree to show all entries - becomes Collapse all after selection

Collapse all

Collapse the tree to show just the top entry - becomes Expand all after selection

Add ESU

Add a new ESU

 

2.If you hove over an existing ESU entry e.g.

Action

Description

Copy ESU Id

Copy the ESU ID to the clipboard

Divide ESU

Divide the ESU at a given point

Unassign from street

Unassign the ESU from the street

Delete

Delete the ESU

 

3.If you select a single ESU from the list you will see a toolbar at the top of the map e.g.

Action

Description

Divide

Divide the ESU at a given point

Unassign

Unassign the ESU from the street

 

4.If you select more than one ESU from the list you will see a slightly different toolbar at the top of the map e.g.

Action

Description

Merge

Merge the selected ESUs

Unassign

Unassign the ESU from the street
