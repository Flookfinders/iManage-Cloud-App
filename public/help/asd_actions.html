The actions for ASD are found when the ASD tab is selected and there are several places where actions are displayed.

1.At the top of the form.

Action

Description

Expand all

Expand the tree to show all entries - becomes Collapse all after selection

Collapse all

Collapse the tree to show just the top entry - becomes Expand all after selection

Add ASD

Clicking this button then displays the different types of ASD record you may add e.g.

Note: The option to add a Public right of way is not displayed as this is only available on a Type 3 street.

 

2.If you hover an existing ASD record e.g.

Action

Description

Delete ASD record

Delete the type of ASD record selected

 

3.If you select an existing ASD record e.g.

Action

Description

Delete ASD record

Delete the type of ASD record selected

Add ASD record

Add a new ASD record of the same type
