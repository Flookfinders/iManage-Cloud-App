(Scotland only)

The property successor cross reference  (Type 30) provides for historical links between properties in the event of splits and merges. This information is displayed when you click on the Successors tab of the Property form. Click on an entry in the list to display the details.

Note: Multiple successor cross reference records can be held for a single BLPU, e.g. where a property has been split into multiple new properties.

Successor

This is the UPRN for the previous property.

Start Date

The date on which this Successor record was created.

End Date

The date on which this Successor record ceased to exist.

Successor key

The unique key for the Successor record which is automatically generated when a successor is created
