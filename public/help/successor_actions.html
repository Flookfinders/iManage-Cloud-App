(Scotland only)

There are a couple of places where actions are displayed for property successor cross references.

1.If you select the Successors tab for a property you have the option to add a successor cross reference for the property e.g.

Action

Description

Add Successor

Add a successor cross reference for the property

 

2.If you hover over a Successor on the Successors tab for a property e.g.

Action

Description

Delete Successor

Delete the successor cross reference for the property
