iManage Cloud provides the tools for creating and maintaining BS7666 compliant Street and Land & Property gazetteers. It has been developed to provide Local Authorities with the necessary tools to create and maintain both Street and Land & Property gazetteers, in accordance with the current government BS7666 standard.

The application comprises various modules:

•Street Gazetteer (LSG / NSG)

•Land & Property Gazetteer (LLPG / NLPG)

•Associated Street Data (ASD) or for Scotland Associated Data (AD)

Control of access to iManage Cloud is managed by the Symphony Security API user “rights” which govern your role. There are two types of user that may be defined for use with iManage Cloud.

Type

Description

User

A standard user responsible for keeping the gazetteers up-to-date and checking for errors, i.e., modifying the gazetteers as changes to roads and properties occur. Maintenance activities form the bulk of the work in complying with the British Standard BS7666

Administrator

An admin user who has the same rights as standard users but are also responsible for maintaining the following in iManage Cloud:

•Authority details

•Property metadata

•Street metadata

•ASD metadata

•Property templates

•Street template

•ASD template

•Lookup tables

•Map layers

Note: The Symphony Security API also controls the type of data that may be viewed / edited such as the access to NSG, NLPG and ASD data.
