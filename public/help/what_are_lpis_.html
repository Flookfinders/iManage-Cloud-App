What are LPIs?

 Top  Previous  Next

The Land & Property Identifier (LPI) is made up of a Primary Addressable Object (PAO), Secondary Addressable Object (SAO) and Street Reference USRN. Properties are usually identified by their street address e.g. 41 High Street. However, if there were a hotel on the property it may also be identified by the name of the hotel, such as "The White Horse". Properties may be assigned an official name [Approved Official LPI], usually the street address as well as "Other" names [Alternative LPI] to make it easier to identify. An LPI must be referenced to a single BLPU.

An LPI contains the following information:

•LPI_KEY (primary key for LPIs)

•Unique Property Reference Number (UPRN) - used to reference to the corresponding BLPU

•Unique Street Reference Number (USRN) – used to reference to corresponding street

•the language of the identifier

•information relating to the current status within the lifecycle of the identifier

•the property/location name and/or number details

•postal information – Postcode / Posttown (where required)

•metadata relating to the maintenance of the record
