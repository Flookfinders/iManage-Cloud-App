What is a BLPU?

 Top  Previous  Next

The Basic Land and Property Unit [BLPU] represents the real world objects recorded within the gazetteer and can be thought of as relating to “properties” or “locations”, such as a physical unit with a dwelling with its own address, property centroid and mapped boundary (provenance). Each Property (BLPU) has a Unique Property Reference Number (UPRN) that identifies a property chosen from a range provided by the National Gazetteer Custodian. All other information about a property, such as addresses, cross references etc. are linked to the BLPU using the UPRN.

The BLPU contains the following information:

•Unique Property Reference Number (UPRN)

•an easting and northing coordinate to describe the location of the object

•information relating to the current status within the lifecycle of the object

•a three tier classification system

•an organisation name (optional)

•Ward and Parish information (optional)

•metadata relating to the maintenance of the record

•a BLPU must be referenced to one or more LPI .
