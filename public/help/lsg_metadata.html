(England & Wales only)

To select your Street metadata, click on Street metadata under General in the Settings. You will see four boxes:

Click in the box to edit the data. Make the changes required and click Done when you are happy or Cancel to return without updating the details.

Gazetteer

Territory

The geographic domain of the gazetteer.

Linked data

A list of the application datasets used to update the Local Street Gazetteer (LSG).

Frequency

The frequency with which the LSG is maintained to the NSG Custodian. Please select the appropriate option from the list.
Custodian

Name

Enter the name for the Organisation or department/function responsible for the compilation and maintenance of the data in the gazetteer that is a DCA participating authority.

UPRN

Enter the Unique Property Reference Number for the Authority Street Custodian location.

Code

Enter the authority code for the Custodian issued by the NSG Custodian.
Miscellaneous

Classification scheme

This is the classification scheme used for all multiple value specified fields e.g. DEC-NSG v8.1. Note: For Scotland this is "Scottish Gazetteer Conventions v4.0" and cannot be changed.

Metadata date

This is the date at which the gazetteer can be considered to be current.

Language

The default language used for the descriptors within the gazetteer. For Scottish customers the Language should be English if no Gaelic language records are present or Gaelic if the gazetteer contains any Gaelic language records. Please refer to the Bilingual Gazetteers topic for more information if required.

Character set

The description of the Character set used for the data when it is exported.
Content

Please use the slider for each entry to select the Percentage of each type of street that are present in your LSG.
