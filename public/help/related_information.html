On both the Property and Street forms there is a Related tab which shows any other Street or Property that is related to the Street or Property you are displaying. You will then see two sub tabs to display related Properties and Streets.

In the following Related properties example we have selected Properties. Where there are children you will see a number representing the number of children and, if you click on the down arrow to expand the list, the children are displayed slightly indented from their parent. If you wish to expand all the entries at once there is an Expand all option at the top right of the Explorer.

You may also select all of the properties by clicking in the box at the top left of the list which can be useful if you wish to edit all of the properties. Please see Actions below for a detailed list of what information may be changed in this way. Any individual properties that you don't want to include, such as the Street Record, may be individually deselected.

If you wish to edit the details for all the children or a property but not the parent itself then you need to select the parent record which will select all of the children and then hold the Ctrl key and deselect the parent. This will leave the children selected but the parent deselected.

You may also use the select box at the top of the list to select all properties and then use the Ctrl key to deselect the properties you don't want to edit.

Actions

If you select or hover over an entry then you get a list of actions you may perform depending upon whether you have selected a Property or a Street.

Change the status of the BLPU & LPIs to historic and set the end date to today for all associated records
