Getting Started

 Top  Previous  Next

The first thing you need to do is to log on to iManage Cloud using the user name and the password you have been provided with.

You will then be sent an email from Symphony Security Authentication with an authentication code that you need to enter here. Note: If you have forgotten your password you need to enter your user name before clicking Forgot password.

Once you have logged on you will be taken to the Home page for iManage Cloud.
