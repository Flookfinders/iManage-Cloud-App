Maintaining BLPU Notes

 Top  Previous  Next

First you need to find the property that you wish to add a note to. You can use the Search to search for the property by UPRN or address. Either select the property from the search results or press enter in the search and select the property from the Explorer. The details for the property are displayed in the Property form.

•Click on the Notes tab.

Add a Note

•Select to Add new Note from one of the Note Actions.

•Change the BLPU Note details as required.

•Once you are happy, click OK to validate your entry.

•Now click Save to update the property.

Edit a Note

•Select the note that you wish to edit.

•Change the BLPU Note details as required.

•When you are happy, click OK to validate your changes.

•Now click Save to update the property.

Delete a Note

•Select the note that you wish to delete.

•Select to Delete Note from one of the Note Actions.

•You will see a confirmation dialog letting you know what will be deleted if you continue e.g.

•Click to delete the note or click CANCEL to return without deleting.

•Now click Save to update the property.
