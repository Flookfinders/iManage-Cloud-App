(England & Wales only)

To select your Property metadata, click on Property metadata under General in the Settings. You will see three boxes:

Click in the box to edit the data. Make the changes required and click Done when you are happy or Cancel to return without updating the details.

Gazetteer

Name

The name of the gazetteer e.g. LLPG for London Borough of Camden. Note: For Scotland this is a fixed list for selection.

Scope

A description of the content of the gazetteer e.g. all basic property units with some land parcels. Note: For Scotland this is "Scottish Gazetteer Conventions" and cannot be changed.

Territory

The geographic domain of the gazetteer.

Linked data

A list of the application datasets used to update the LLPG.

Owner

The organisation with overall responsibility for the gazetteer.

Frequency

The frequency with which the LLPG is maintained to the NLPG hub. Please select the appropriate option from the list.
Custodian

Name

The name for the Custodian

UPRN

The Unique Property Reference Number for the Custodian.

Code

The authority code for the Custodian
Miscellaneous

Classification scheme

This is the classification scheme used for all multiple value specified fields e.g. NLPG conventions document v1.0. Note: For Scotland this is "Scottish Gazetteer Conventions v4.0" and cannot be changed.

Metadata date

This is the date at which the gazetteer can be considered to be current.

Language

The default language used for the descriptors within the gazetteer. For Scottish customers the Language should be English if no Gaelic language records are present or Gaelic if the gazetteer contains any Gaelic language records. Please refer to the Bilingual Gazetteers topic for more information if required.

Character set

The description of the Character set used for the data when it is exported.
