Street template

 Top  Previous  Next

The Street template is where you may set the defaults for creating streets. Setting up defaults for details that will predominantly be the same for every entry speeds up the data entry process considerably. Setting default values does not mean that these cannot be overridden at the time of creating the data it simply populates the initial dialog with the defaults.

To maintain your Street template, click on Street template under Customisation in the Settings. There are four boxes displayed for the different areas of the street.

1.Street defaults
Details associated with the actual Street (Type 11) that appear on the Street tab of the Street form. Please see Street details for a description of each of the entries.

2.ESU defaults
Details associated with the ESUs (Type 13) that have been assigned to the street such as Direction, and appear on the ESU tab of the Street form. Please see ESU details for a description of each of the entries.

3.One-way Exemption defaults
Details associated with One-way Exemptions (Type 16) that are used to supply exemptions to a One-way direction for an ESU, and appear on the ESU tab of the Street form. Please see One-way Exemption details for a description of each of the entries.

4.Highway Dedication defaults
Details associated with Highway Dedications (Type 17) that hold a description of who is allowed to use the street for each ESU section, and appear on the ESU tab of the Street form. Please see Highway Dedication details for a description of each of the entries.

Click on a box to edit the details and click Done when finished or Cancel to return without updating the details.
