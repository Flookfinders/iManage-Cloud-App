In iManage Cloud, the graphics for basic land and property units (BLPUs) are all stored in the SQL database. There is a table for BLPU points (representing the centroid of each parcel) and a table for BLPU provenance boundaries (delineating parcel boundaries, land uses, building boundaries etc). Attribute data about BLPUs, such the street address (LPI), alternative common names and cross-references, are stored in the database and can be edited using Property form.

Note: You may perform more than one function when editing information for a particular property before clicking on the Save button. E.g. Editing the LPI, creating a new cross-reference, adding a BLPU Provenance polygon etc.
