Map Layer details

 Top  Previous 

These are the details that are required for display a Map layer in the iManage Cloud Map pane. They may be edited by an Administrator clicking on the Map layers under Customisation in the Settings.

Note: Some of the entries below are dependent on the service Type.

Title

The title of the layer which is used to identify the layer in the Layer control.

Provider

Select the service provider for the layer from the supplied list.

Type

Select the type of service to use to retrieve the layer from the provider.

URL

The URL to the service from which you wish to retrieve your map layer.

Layer

The layer ID of the layer you wish to retrieve from the service.

Property names

WFS only

The comma separated list of property names you wish to be returned by the WFS.

Key

WMTS only

The API Key you have been given to access the service.

Mode

WMTS only

The mode to used to return the data from the service. If you are using WMTS from Ordnance Survey then this would be KVP (Key Value Pair).
Geometry type

Select the type geometry type for the layer from the supplied list.

Map id

The unique id assigned to the layer you wish to retrieve.

Copyright information

The copyright information you would like displayed on the map in iManage Cloud. If you wish to display the current year then you may use <<year>>.

Use paging

WFS only

If you wish to use paging to return the records then set this to Yes.

Max. batch size

WFS only

The maximum number of records to return in each batch.

Active

If you wish to use the layer in iManage Cloud then this must be set to Yes. Select No if wish to keep the layer in the Map layers list but not display it in the Map pane.

Display in layer control

If you wish the layer to be displayed in the Layer Control then set this to Yes. The user will then have the option to Show / Hide the Layer or change its opacity in the Map pane. If you do not wish the user to Show / Hide the layer then set this to No.

Scale range

The minimum scale (most zoomed out) and maximum scale (most zoomed in) at which the layer is visible in the Map pane.

Opacity

The opacity of the layer in the Map pane where 0 = Transparent and 1 = Opaque.

ESU snap to layer

WFS only

Set this to Yes if you wish to use this layer to snap ESU nodes to when editing ESU geometry.

BLPU snap to layer

WFS only

Set this to Yes if you wish to use this layer to snap Property centroids to when positioning properties.

Extent snap to layer

WFS only

Set this to Yes if you wish to use this layer to snap BLPU Extent (Provenance boundary) nodes to when editing Extents geometry.
