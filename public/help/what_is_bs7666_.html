What is BS7666?

 Top  Previous  Next

BS7666 is a British Standard that sets out the guidelines upon which the NLPG is built. It contains standards by which to record Basic Land and Property Units (BLPUs), which include within them a Unique Property Reference Number (UPRN), grid co-ordinates and one or more Land Property Identifiers (LPIs).

It is BS7666 standardisation that enables the amalgamation of all local authority address data into the NLPG and is the standard that makes accurate and efficient address management possible.

BS7666 - Addressable Objects

BS7666 lays down guidelines for the creation and management of address objects, which are real world objects that have fixed locations and may be identified and referenced by one or more addresses.

Addressable Objects are divided into two categories:

•Primary Addressable Objects (PAO), which is the name given to an addressable object that does not require referencing to another addressable object e.g. house number or building name.

•Secondary Addressable Objects (SAO), which is the name given when an addressable object is referenced to a PAO e.g. Flat 3 (SAO), Wellington House (PAO).

BS7666 - Non-Addressable Objects

Unlike many other address databases, the BS7666 standard enables the creation and management of non-addressable objects such as communication masts. These non-addressable objects form an essential component within the NLPG, and through the BS7666 standard, they can be held alongside postal addresses. This means they can be accessed simultaneously, cross referenced etc.

BS7666 - Precise, consistent formatting

The exact formatting that is derived through BS7666 means that the components that make up an address will be consistent throughout. This makes division by postcode, county locality or postal town quick, easy and accurate.
