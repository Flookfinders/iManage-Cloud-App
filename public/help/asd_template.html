The ASD template is where you may set the defaults for creating ASD. Setting up defaults for details that will predominantly be the same for every entry speeds up the data entry process considerably. Setting default values does not mean that these cannot be overridden at the time of creating the data it simply populates the initial dialog with the defaults.

To maintain your ASD template, click on ASD template under Customisation in the Settings.

England & Wales

For authorities in England and Wales there are five boxes displayed for the different areas of ASD.

1.Interested Organisation defaults
Details for Interest records (Type 61) which are used to supply information on organisations who have an interest in the street, and appear on the ASD tab of the Street form. Please see Interested Organisation details for a description of each of the entries.

2.Construction defaults
Details for Construction records (Type 62) which are used to supply reinstatement categories for a street, and appear on the ASD tab of the Street form. Please see Construction details for a description of each of the entries.

3.Special Designation defaults
Details for Special Designation records (Type 63) which are used to supply special designations that apply to a street, and appear on the ASD tab of the Street form. Please see Special Designation details for a description of each of the entries.

4.Height, Width, Weight Restriction defaults
Details for any Height Width & Weight Restriction [HWW] records (Type 64) which are used to supply restrictions for a street such as a weight restriction, and appear on the ASD tab of the Street form. Please see Height Width and Weight Restriction details for a description of each of the entries.

5.Public Right of Way defaults
Details for any Public Right of Way (PRoW) record (Type 66) which are used to supply the PRoW details for a Type 3 Street, and appear on the ASD tab of the Street form. Please see Public Right of Way details for a description of each of the entries.

Click on a box to edit the details and click Done when finished or Cancel to return without updating the details.

Scotland

For authorities in Scotland there are three boxes displayed for the different areas of Associated Data.

1.Maintenance responsibility defaults
Details for Maintenance responsibility records (Type 51) which are used to supply information relating to the authority or authorities responsible for maintaining a road, and appear on the AD tab of the Street form. Please see Maintenance Responsibility details for a description of each of the entries.

2.Reinstatement category defaults
Details for Reinstatement category records (Type 52) which are used to supply information relating to the reinstatement category of a road or of particular sections or elements of a road, and appear on the AD tab of the Street form. Please see Reinstatement Category details for a description of each of the entries.

3.Special Designation defaults
Details for Special Designation records (Type 53) which are used to supply data relating to special designations applying to the road in general; to sections of the road or to specific points on the road, and appear on the AD tab of the Street form. Please see Special Designation details for a description of each of the entries.

Click on a box to edit the details and click Done when finished or Cancel to return without updating the details.
