There are several places where actions are displayed for LPIs.

1.If you hover over an LPI on the Details tab e.g.

Action

Description

Copy address

Copy the address to the clipboard

Add new LPI

Add an LPI to the property

Delete LPI

Delete the LPI

 

2.If you select the LPI from the Details tab and click Actions e.g.

Action

Description

Add new LPI

Add an LPI to the property

Copy address

Copy the address to the clipboard

Reject LPI

Change the status of the LPI to rejected

Historicise LPI

Change the status of the LPI to historic

Delete LPI

Delete the LPI
