Maintaining Street Notes

 Top  Previous  Next

First you need to find the street that you wish to add a note to. You can use the Search to search for the street. Either select the street from the search results or press enter in the search and select the street from the Explorer. The details for the street are displayed in the Street form.

•Click on the Notes tab.

Add a note

•Select to Add new note from one of the Note Actions.

•Change the details as required.

•Once you are happy, click OK to validate your entry.

•Now click Save to update the street.

For a complete description of all the entries please read the Street note details topic.

Edit a note

•Select the note that you wish to edit.

•Change the Street note details as required.

•When you are happy, click OK to validate your changes.

•Now click Save to update the street.

Delete a note

•Select the note that you wish to delete.

•Select to Delete note from one of the Note Actions.

•You will see a confirmation dialog letting you know what will be deleted if you continue e.g.

•Click to delete the note or click CANCEL to return without deleting.

•Now click Save to update the street.
