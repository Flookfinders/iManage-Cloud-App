Authority details

 Top  Previous  Next

To select your Authority details, click on the Authority details under General in the Settings.

Click in the box to edit the data. Make the changes required and click Done when you are happy or Cancel to return without updating the details.

Name

Enter the name of your Authority.

Code

The code that is defined as the DETR Data Capture Code for the Authority it is a 4 digit number.

MSA text

This entry is compulsory and should contain the text that you wish to display as the copyright statement in the Map pane. This should include the licence number for the authority to access Ordnance Survey's digital mapping through the Mapping Services Agreement (MSA).

Tab text

The text you enter here is used on the browser tab for iManage Cloud. If you leave this blank the default is iManage Cloud (Authority_Name) where Authority_Name is name of the Authority taken from the DETR code you have entered above.

USRN range

The range of numbers for the Unique Street Reference Numbers (Street Identifiers) to be allocated to any new Streets. This USRN range is allocated to you by the National Hub.

Clicking on the Update button will allow you to enter another "New" range of numbers to update the AVAILABLE_USRNS table. This is the table that is used to assign a USRN when you create a new street.

UPRN range

The range of numbers for the Unique Property Reference Numbers to be allocated to any new Properties. This UPRN range is allocated to you by the National Hub.

Clicking on the Update button will allow you to enter an entire "New" range of numbers to update the AVAILABLE_UPRNS table. This is the table that is used to assign a UPRN when you create a new property.

Create street BLPU

If you wish the Street BLPU record to be automatically created when you create a new street then select Yes. If left as No you will need to manually create this record for a new street.

Display language

All information will be displayed in English unless you are a dual language authority e.g. Welsh; if this is the case you can decide which language you wish to be displayed by default.

Address fields uppercase

Selecting Yes will automatically force all the address text fields to be uppercase, leaving as No will leave the entries as you type them.

Note 1: This will only force any text that you enter / change to be uppercase, it will not change any existing entries.

Note 2: When performing an NSG export all records will be exported in uppercase
