(Scotland only)

To select your Gazetteer metadata, click on Gazetteer metadata under General in the Settings. You will see three boxes:

Click in the box to edit the data. Make the changes required and click Done when you are happy or Cancel to return without updating the details.

Gazetteer

Name

This can be one of the following depending upon the type of data held in your gazetteer.

•Land and Property Gazetteer

•Street Gazetteer

•Associated Street Data

•Street Gazetteer and Associated Street Data

•Land and Property Gazetteer, Street Gazetteer & Associated Street Data

Scope

This must be "Scottish Gazetteer Conventions" and cannot be changed.

Territory

The local authority area e.g. Falkirk.

Owner

The local authority area (official title) with overall responsibility for the gazetteer e.g. Falkirk Council.
Custodian

Name

The gazetteer Custodian contact email address.

Code

The authority code for the gazetteer Custodian.
Miscellaneous

Classification scheme

This must be "Scottish Gazetteer Conventions v4.0" and cannot be changed.

State code scheme

This must be "Scottish Gazetteer Conventions v4.0" and cannot be changed.

Metadata date

This is the date the metadata was last updated.

Gazetteer date

This is the date at which the copy of the gazetteer was generated.

Language

The Language should be English if no Gaelic language records are present or Gaelic if the gazetteer contains any Gaelic language records. Please refer to the Bilingual Gazetteers topic for more information if required.
